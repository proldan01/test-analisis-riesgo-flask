How to read this chart

Each point represents a randomly generated portfolio. The color gradient shows the Sharpe ratio — green = high risk-adjusted return, red = low.

The ★ star marks the Maximum Sharpe portfolio — optimal under Modern Portfolio Theory.
The ◆ diamond marks the Minimum Volatility portfolio — most conservative allocation.
